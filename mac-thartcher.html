What is a Website?

A website is a collection of web pages and related content that is identified by a common domain name and published on at least one web server. Examples of websites include Wikipedia, Amazon, and YouTube.

What is a Server?

A server is a computer or system that provides resources, data, or services to other computers or systems over a network. In the context of the web, servers host websites and respond to requests from web browsers.

What is Hosting?

Hosting refers to the service of providing storage space and access for websites on a server. Web hosting companies offer various plans that allow individuals and organizations to make their websites accessible on the internet.

IP Addresses and URLs

IP stands for "Internet Protocol"

It is a unique address (eg 192.185.1.38)that identifies a device so other machines can find it.

URL stands for "Uniform Resource Locator"is the web address used to locate a resource

The Resolver talks to the Root, TLD, and Authoritative servers.

How does the DNS work?

DNS stands for "Domain Name System". It's like asking a librarian for a book:

  1. You type the name (Domain).
  2. The Browser asks the DNS Resolver for the "address".
  3. Once it gets the IP, it finds the server and shows you the site.
