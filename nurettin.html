How CSS selects HTML elements?

Element selector

selects all HTML elements of a specific type


						p { 
							color: red;
						}

Class Selector

selects all elements that have a specific class attribute


						.box {
							background-color:blue;
						}

ID Selector

selects a single unique element that has a specific id attribute


						#header {
							background-color:gray;
						}

Pseudo-class Selector

defines a special state of an element

(Hover, Active, Visited)


						a:hover {
							color: white;
							background-color: blue;

						}
